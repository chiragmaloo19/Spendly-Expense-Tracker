# Spendly — Personal Finance Tracker

Spendly is a premium, high-performance web application designed to help users take full control of their personal finances. With a focus on speed, aesthetics, and clear data visualization, Spendly makes tracking income and expenses effortless and insightful.

## 🚀 Overview
Managing money shouldn't be a chore. Spendly provides a sleek, glassmorphic interface that gives you an immediate high-level summary of your financial health while allowing for deep dives into your spending patterns. Built using modern Vanilla JavaScript, it offers a seamless experience without the need for complex frameworks or backend setups.

## ✨ Features
- **Comprehensive Tracking**: Seamlessly log both expenses and income with specific categories, dates, and descriptions.
- **Dynamic Dashboard**:
  - **Net Balance**: See your current financial standing at a glance.
  - **Monthly Insights**: Automatically calculates spending for the current month.
  - **Biggest Expense**: Quickly identify your largest single expenditure.
  - **Transaction Count**: Keep track of the volume of your financial activity.
- **Visual Analytics**:
  - **Expense Breakdown**: A dynamic Doughnut chart showing spending distribution by category.
  - **Historical Comparison**: A multi-colored Bar chart comparing Monthly Income vs. Expenses over the last 6 months.
- **Advanced Management**:
  - **Filtering**: Sort through transactions by type (Income/Expense) or specific categories.
  - **CSV Export**: Download your entire transaction history as a professionally formatted CSV file with a single click.
- **Data Persistence**: Uses `LocalStorage` to ensure your data remains safely stored in your browser across sessions.
- **Responsive Design**: Fully optimized for desktops and tablets with a premium, modern aesthetic using the Inter font family.

## 🛠️ Technology Used
- **Logic**: Vanilla JavaScript (ES6+), LocalStorage API
- **Structure**: Semantic HTML5
- **Styling**: Modern CSS3 (Grid, Flexbox, Custom Properties)
- **Charts**: [Chart.js](https://www.chartjs.org/) for high-performance data visualization
- **Typography**: Google Fonts (Inter)
- **Icons**: Unicode Symbols & Emojis

## 📊 Output
The application consists of a dual-column layout:
1. **Left Column**: Features a tabbed form for quick entry and a scrollable, filtered transaction history list with elegant status badges.
2. **Right Column**: Houses the analytical dashboard with interactive charts that update in real-time as you add or delete entries.
3. **Toast Notifications**: Provides instant feedback for actions like "Added", "Deleted", or "Exported".
